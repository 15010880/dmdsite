폴더 구조


bin : app.js에서 실행시키는 파일이 들어있음

db : db와 관련된 파일 ---제작중

node_modules : package.json 에 쓰여져있는 모듈들을 다 다운받는 폴더
		
public : 자바스크립트와 css,이미지를 넣는 폴더(기능적인 폴더)

routes : views 에 있는 파일들과 app.js를 연결해주는 파일들을 넣어놓은 폴더(통로)

views : 화면에 보여지는 전체적인 틀을 잡아놓은 파일들(ejs,html)(뷰어) 

package.json : 지금까지 쓴 모듈들과 버전정보등 프로젝트 전반적인 내용이 담겨져 있는 파일
		모듈을 선언하면 자동으로 이 파일에 쓰이게되며 
		npm install 을 사용하면 이 파일을 기반으로 모듈들이 node_modules 폴더에 깔리게된다

app.js : 모듈을 선언하고 여러가지 파일들을 묶어주는 파일
	app.js ---> /bin/www 에 있는 파일을 실행

실행 원리
views 에 있는 파일들로 화면에 전체적인 뼈대를 잡고 public 에있는 기능파일(javascript,css,image)을 이용해 한 페이지를 구성하게 된다
그 뒤 routes를 통해 app.js 파일로 각각의 views에 있는 파일들을 묶어 한번에 실행

public -> views -> routes -> app.js -> /bin/www
